# Python Introduction Exercises

This repository contains a set of introductory Python exercises to practice basic programming concepts including variables, arithmetic operations, and script execution.

---

## Project Overview

This project demonstrates the use of Python for simple calculations and variable assignments. The exercises include:

- Correct variable assignment for `number1` and `number2`.
- Use of arithmetic operators for addition, subtraction, multiplication, division, modulus, and exponentiation.
- Explicit `print` statements showing the results of each operation.
- No use of global variables.
- Code runs without errors.

---

## Files Included

- `basic_operations.py` : Contains the main Python script with all arithmetic operations.
- `future_age_calculator.py` : Additional exercise file (if applicable).
- `README.md` : This documentation file.
